Struts2 Web App

Intent

To create an application that uses:
  • Struts2 MVC Framework
  • Fully functional front-end using JQuery/Boostrap Javascript
  • Hibernate ORM for object to relational DB mapping for persistence
  • Maven for building and app deployment for testing
  • Docker file for a self-contained image

Usage

  • Jetty

    To quickly execute the application, issue the following from the app root directory- mvn jetty:run

  • Docker

    Build: docker build -t namespace/repository:version .
    namespace - your_company_name
    repository - your_app_name
    version - latest

    The '.' at the end of the build command indicates to use the Dockerfile and content of the current directory

    Run:vThe following usage of docker run assumes MySQL is in use by the app and is listening on port 3306

    docker run -it -d -p 52000:3306 -p 8080:8080 /:

    52000 is used by way of example for the purpose of avoiding a port conflict in the event MySQL is being run locally on the host machine.
