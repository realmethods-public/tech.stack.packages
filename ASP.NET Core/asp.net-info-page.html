Microsoft ASP.NET Tech Stack

Intent

To create an application that uses:
  • ASP.NET Core 2.0 and .NET Framework v4x for full MVC support
  • Fully functional front-end using .cshtml with embedded JQuery and Bootstrap
  • nHibernate for model to table db mapping
  • GitHub to push application files

Usage

Step 1

From Visual Studio, create an ASP.NET Core 2.0 project that uses .NET Framework. (not the .NET Core)
Step 2
Use NuGet within Visual Studio to load MySQL and nHibernate.
Step 3
Run cloudMigrate to create an app and save the app to a directory using the same name as the VS project.
Step 4
Download from GitHub (or copy) the cloudMigrate files overwriting into the Visual Studio project normally located at

C:\Users\\source\repos\
Step 5
Notice if Visual Studio auto-refreshes the project with the goFramework created application files. If the project does not auto-refresh, force it to. If the files still do not appear, re-check they were copied into the correct directory.
Step 6
Locate the hibernate.cfg.xml fle, right-click it and assign Build Action to Embedded Resource and Copy to Output to Copy Always.
Step 7
Build then run the project. Visual Studio will launch the app in the browser.
